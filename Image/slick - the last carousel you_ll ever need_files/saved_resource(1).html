                                                                                                                            In addition to setting variableWidth: true, instead of .slick-slide{ width: 270px;} you should use .slick-slide img {margin-right: 25px} (or whatever you want your right margin to be). This will ensure the same amount of space between images no matter what the width of each image is. Just setting a hard width value basically overrides the intention of variableWidth: true.

                                                                                                                                                                    The arrows show up as white, so if your page is white you wont see them. I also had this problem. Change the body to black and you'll see them. Alternatively to edit the styles of the arrows, assuming you arent using their "slick-theme.css" and only using "slick.css",(also if you're using font awesome for the icons), in the settings set:

                                                                                                                                                                    nextArrow:$('.next'),
                                                                                                                                                                    prevArrow:$('.prev'),

                                                                                                                                                                    then place your html elements where you want:

                                                                                                                                                                    <div class="next fas fa-chevron-right"></div>
                                                                                                                                                                    <div class="prev fas fa-chevron-left"></div>

                                                                                                                                                                    then you can edit them in css:

                                                                                                                                                                    .next{
                                                                                                                                                                    position:absolute;
                                                                                                                                                                    right:0;
                                                                                                                                                                    top:50%;
                                                                                                                                                                    tranform:translateY(-50%);
                                                                                                                                                                    etc... whatever you want to style them as.
                                                                                                                                                                    }

                                                                                                                                                                    Similarly for the dots you just make a container element for them. In the settings:

                                                                                                                                                                    appendDots:$('.dots-wrapper'),

                                                                                                                                                                    then in html place your element where you want

                                                                                                                                                                    <div class="dots-wrapper"></div>

                                                                                                                                                                    ... And don't forget the period when selecting the class with jQuery... Shit had me stumped for a half hour.
